- ## 光标移动命令
	- 好的，在 Ubuntu（以及绝大多数 Linux 发行版）的终端（Terminal）中，熟练使用快捷键可以极大地提高您的工作效率。这些快捷键主要基于 **Readline** 库，其键位绑定风格与著名的文本编辑器 **Emacs** 非常相似。
	  
	  以下是终端中光标移动和编辑的常用命令，我将它们分类以便于记忆。
	- ### ⭐ 必知必会（强烈推荐）
		- 这几个是日常使用频率最高的，建议优先掌握。
		  
		  | 快捷键 | 功能 | 说明 |
		  | :--- | :--- | :--- |
		  | **`Ctrl + A`** | 移动到行首 | 相当于 `Home` 键，非常快速。 |
		  | **`Ctrl + E`** | 移动到行尾 | 相当于 `End` 键，与 `Ctrl+A` 绝配。 |
		  | **`Ctrl + R`** | 反向搜索历史命令 | 按下后输入关键字，会立即查找包含该关键字的最近命令。 |
		  | **`Ctrl + L`** | 清屏 | 效果等同于输入 `clear` 命令并回车。 |
		  | **`Ctrl + C`** | 终止当前命令 | 强制停止当前正在运行的前台程序。 |
		  | **`Ctrl + U`** | 删除从光标到行首的内容 | 快速清除光标左边的所有字符。 |
		  | **`Ctrl + K`** | 删除从光标到行尾的内容 | 快速清除光标右边的所有字符。 |
		  
		  ---
	- ### 一、字符级移动
		- | 快捷键 | 功能 | 说明 |
		  | :--- | :--- | :--- |
		  | **`Ctrl + F`** | 向前移动一个字符 | F = Forward，等同于 `→` (右箭头)。 |
		  | **`Ctrl + B`** | 向后移动一个字符 | B = Backward，等同于 `←` (左箭头)。 |
	- ### 二、单词级移动
		- 这对于快速在长命令中跳转非常有用。
		  
		  | 快捷键 | 功能 | 说明 |
		  | :--- | :--- | :--- |
		  | **`Alt + F`** | 向前移动一个单词 | F = Forward，光标跳到下一个单词的词首。 |
		  | **`Alt + B`** | 向后移动一个单词 | B = Backward，光标跳到当前或上一个单词的词首。 |
		  
		  > **提示**：在某些终端模拟器或SSH客户端中，`Alt` 键可能无法正常工作。如果 `Alt + F` 不起作用，可以尝试先按 `Esc` 键，然后立即按 `F` 键，通常能达到同样的效果。
	- ### 三、编辑与删除
		- | 快捷键 | 功能 | 说明 |
		  | :--- | :--- | :--- |
		  | **`Backspace`** | 删除光标前的一个字符 | |
		  | **`Ctrl + H`** | 删除光标前的一个字符 | 功能同 `Backspace`。 |
		  | **`Ctrl + D`** | 删除光标下的字符 | 功能同 `Delete` 键。如果当前行为空，则会退出终端会话。 |
		  | **`Ctrl + W`** | 删除光标前的一个单词 | |
		  | **`Alt + D`** | 删除光标后的一个单词 | |
		  | **`Ctrl + Y`** | 粘贴（Yank） | 粘贴最后一次用 `Ctrl+U`, `Ctrl+K` 或 `Ctrl+W` 删除的内容。 |
		  | **`Ctrl + T`** | 交换光标前后字符 | T = Transpose，很有趣，可以快速修正打字颠倒的错误。 |
		  | **`Ctrl + _`** (下划线) | 撤销上一次操作 | 这是一个撤销（Undo）命令，可以连续按。 |
	- ### 四、历史命令操作
		- | 快捷键 | 功能 | 说明 |
		  | :--- | :--- | :--- |
		  | **`↑` (上箭头)** | 上一个历史命令 | |
		  | **`Ctrl + P`** | 上一个历史命令 | P = Previous。 |
		  | **`↓` (下箭头)** | 下一个历史命令 | |
		  | **`Ctrl + N`** | 下一个历史命令 | N = Next。 |
		  | **`!!`** (输入) | 执行上一条命令 | 输入 `!!` 然后回车。 |
		  | **`!关键字`** (输入) | 执行最近一条以“关键字”开头的命令 | 例如，`!sudo` 会执行最近一条以 `sudo` 开头的命令。 |
		  | **`Alt + .`** (点) | 插入上一条命令的最后一个参数 | 非常实用！例如，你先 `mkdir my_new_dir`，然后想进入它，只需输入 `cd` 再按 `Alt + .` 即可补全为 `cd my_new_dir`。 |
	- ### 五、进程控制
		- | 快捷键 | 功能 | 说明 |
		  | :--- | :--- | :--- |
		  | **`Ctrl + Z`** | 暂停当前命令 | 将当前前台进程放入后台并暂停，可使用 `fg` 命令恢复。 |
	- ### 总结与建议
		- 刚开始可能记不住这么多，但不必担心。建议从 **“必知必会”** 部分开始，有意识地在日常操作中使用它们：
		  
		  1.  用 **`Ctrl + A`** 和 **`Ctrl + E`** 在行首行尾快速跳转。
		  2.  用 **`Ctrl + R`** 搜索历史命令，告别反复按上箭头。
		  3.  输入错误时，用 **`Ctrl + U`** 或 **`Ctrl + K`** 快速清空部分或全部内容。
		  4.  需要清屏时，用 **`Ctrl + L`** 代替 `clear`。
		  
		  一旦这些形成了肌肉记忆，你就会发现自己在命令行下的操作效率有了质的飞跃。
